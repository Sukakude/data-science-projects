# Rice type classification

### Project Overview
This project focuses on building a deep learning model to classify different types of rice grains using image data. 
The model is implemented using PyTorch and trained on the dataset from Rice Type Classification Dataset available on Kaggle.

This project demonstrates skills in:
- Deep Learning
- PyTorch
- Data preprocessing
- Model training & evaluation

---

# Dataset

The dataset used in this project comes from Kaggle:
https://www.kaggle.com/datasets/mssmartypants/rice-type-classification
